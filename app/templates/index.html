📖 API Documentation

      

        
All /api/* endpoints require an API key. Pass it via Authorization: Bearer <key> or X-API-Key: <key>.


        
Authentication

        

          

            POST/api/loginDashboard login
          

          

            
Authenticate to the dashboard UI. Sets a session cookie.

            
curl -X POST http://localhost:5001/api/login \
  -H "Content-Type: application/json" \
  -d '{"password": "admin"}'

          

        


        
Customers

        

          

            GET/api/customersList all customers
          

          

            
Returns all customers with order counts. Optional ?search=term filter.

            
curl http://localhost:5001/api/customers \
  -H "Authorization: Bearer fsk_YOUR_KEY"

# With search
curl "http://localhost:5001/api/customers?search=john" \
  -H "Authorization: Bearer fsk_YOUR_KEY"

            
Response: Array of customer objects with order_count.

          

        

        

          

            GET/api/customers/:idGet a customer
          

          

            
curl http://localhost:5001/api/customers/UUID \
  -H "Authorization: Bearer fsk_YOUR_KEY"

          

        

        

          

            POST/api/customersCreate a customer
          

          

            
curl -X POST http://localhost:5001/api/customers \
  -H "Authorization: Bearer fsk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "firstname": "Jane",
    "lastname": "Smith",
    "email": "jane@example.com",
    "username": "janes",
    "phone": "555-0100",
    "city": "Portland",
    "street": "Oak Ave",
    "number": 42,
    "zipcode": "97201"
  }'

          

        

        

          

            PUT/api/customers/:idUpdate a customer
          

          

            
curl -X PUT http://localhost:5001/api/customers/UUID \
  -H "Authorization: Bearer fsk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"email": "newemail@example.com", "city": "Seattle"}'

            
Send only the fields you want to update.

          

        

        

          

            DELETE/api/customers/:idDelete a customer (and their orders)
          

          

            
curl -X DELETE http://localhost:5001/api/customers/UUID \
  -H "Authorization: Bearer fsk_YOUR_KEY"

          

        

        

          

            GET/api/customers/:id/ordersGet customer's orders
          

          

            
curl http://localhost:5001/api/customers/UUID/orders \
  -H "Authorization: Bearer fsk_YOUR_KEY"

          

        


        
Orders

        

          

            GET/api/ordersList all orders
          

          

            
Returns all orders with items and customer info. Optional ?search=term to filter by customer name or order number.

            
curl http://localhost:5001/api/orders \
  -H "Authorization: Bearer fsk_YOUR_KEY"

curl "http://localhost:5001/api/orders?search=john" \
  -H "Authorization: Bearer fsk_YOUR_KEY"

          

        

        

          

            GET/api/orders/:idGet an order
          

          

            
curl http://localhost:5001/api/orders/UUID \
  -H "Authorization: Bearer fsk_YOUR_KEY"

          

        

        

          

            POST/api/ordersCreate an order
          

          

            
curl -X POST http://localhost:5001/api/orders \
  -H "Authorization: Bearer fsk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_id": "UUID",
    "date": "2024-07-01T12:00:00",
    "products": [
      {"product_id": 1, "quantity": 2},
      {"product_id": 5, "quantity": 1}
    ]
  }'

          

        

        

          

            PUT/api/orders/:id/customerReassign order to different customer
          

          

            
curl -X PUT http://localhost:5001/api/orders/ORDER_UUID/customer \
  -H "Authorization: Bearer fsk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"customer_id": "NEW_CUSTOMER_UUID"}'

          

        

        

          

            DELETE/api/orders/:idDelete an order
          

          

            
curl -X DELETE http://localhost:5001/api/orders/UUID \
  -H "Authorization: Bearer fsk_YOUR_KEY"

Batch & Admin

        

          

            POST/api/customers/batchBatch create customers
          

          

            
curl -X POST http://localhost:5001/api/customers/batch \
  -H "Authorization: Bearer fsk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"count": 10}'

          

        

        

          

            POST/api/orders/batchBatch create orders
          

          

            
# Random customers
curl -X POST http://localhost:5001/api/orders/batch \
  -H "Authorization: Bearer fsk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"count": 5}'

# Linked to specific customers
curl -X POST http://localhost:5001/api/orders/batch \
  -H "Authorization: Bearer fsk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"count": 3, "link_customer_ids": ["UUID1", "UUID2"]}'

          

        

        

          

            GET/api/statsDatabase statistics
          

          

            
curl http://localhost:5001/api/stats \
  -H "Authorization: Bearer fsk_YOUR_KEY"

          

        


        
API Key Management

        

          

            POST/api/keysGenerate a new API key
          

          

            
curl -X POST http://localhost:5001/api/keys \
  -H "X-Master-Key: YOUR_MASTER_KEY" \
  -H "Content-Type: application/json" \
  -d '{"label": "my-agent"}'

          

        

        

          

            GET/api/keysList all API keys
          

          

            
curl http://localhost:5001/api/keys \
  -H "X-Master-Key: YOUR_MASTER_KEY"

          

        

        

          

            POST/api/keys/:prefix/revokeRevoke an API key
          

          

            
curl -X POST http://localhost:5001/api/keys/fsk_abcde/revoke \
  -H "X-Master-Key: YOUR_MASTER_KEY"
