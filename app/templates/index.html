📖 API Documentation

      

        
All /api/* endpoints require an API key. Pass it via Authorization: Bearer <key> or X-API-Key: <key>.


        
Authentication

        

          

            POST/api/loginDashboard login
          

          

            
Authenticate to the dashboard UI. Sets a session cookie.

            
curl -X POST http://localhost:5001/api/login \
  -H "Content-Type: application/json" \
  -d '{"password": "admin"}'

          

        


        
Customers

        

          

            GET/api/customersList all customers
          

          

            
Returns all customers with order counts. Optional ?search=term filter. Email is the primary key.

            
curl http://localhost:5001/api/customers \
  -H "Authorization: Bearer ***"

          

        

        

          

            GET/api/customers/:emailGet a customer by email
          

          

            
curl http://localhost:5001/api/customers/john@gmail.com \
  -H "Authorization: Bearer ***"

          

        

        

          

            POST/api/customersCreate a customer
          

          

            
curl -X POST http://localhost:5001/api/customers \
  -H "Authorization: Bearer ***" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "jane@example.com",
    "firstname": "Jane",
    "lastname": "Smith",
    "username": "janes",
    "phone": "555-0100",
    "city": "Portland",
    "street": "Oak Ave",
    "number": 42,
    "zipcode": "97201"
  }'

          

        

        

          

            PUT/api/customers/:emailUpdate a customer
          

          

            
curl -X PUT http://localhost:5001/api/customers/jane@example.com \
  -H "Authorization: Bearer ***" \
  -H "Content-Type: application/json" \
  -d '{"city": "Seattle"}'

          

        

        

          

            DELETE/api/customers/:emailDelete a customer (and their orders)
          

          

            
curl -X DELETE http://localhost:5001/api/customers/jane@example.com \
  -H "Authorization: Bearer ***"

          

        

        

          

            GET/api/customers/:email/ordersGet customer's orders
          

          

            
curl http://localhost:5001/api/customers/john@gmail.com/orders \
  -H "Authorization: Bearer ***"

          

        


        
Orders

        

          

            GET/api/ordersList all orders
          

          

            
Returns all orders with items and customer info. Order number is a 5-digit primary key. Optional ?search=term to filter by customer name, email, or order number.

            
curl http://localhost:5001/api/orders \
  -H "Authorization: Bearer ***"

          

        

        

          

            GET/api/orders/:order_numberGet an order by number
          

          

            
curl http://localhost:5001/api/orders/10001 \
  -H "Authorization: Bearer ***"

          

        

        

          

            POST/api/ordersCreate an order
          

          

            
curl -X POST http://localhost:5001/api/orders \
  -H "Authorization: Bearer ***" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_email": "john@gmail.com",
    "date": "2024-07-01T12:00:00",
    "shipped_date": "2024-07-03",
    "delivery_date": "2024-07-08",
    "products": [
      {"product_id": 1, "quantity": 2},
      {"product_id": 5, "quantity": 1}
    ]
  }'

          

        

        

          

            PUT/api/orders/:order_number/customerReassign order to different customer
          

          

            
curl -X PUT http://localhost:5001/api/orders/10001/customer \
  -H "Authorization: Bearer ***" \
  -H "Content-Type: application/json" \
  -d '{"customer_email": "jane@example.com"}'

          

        

        

          

            DELETE/api/orders/:order_numberDelete an order
          

          

            
curl -X DELETE http://localhost:5001/api/orders/10001 \
  -H "Authorization: Bearer ***"

Batch & Admin

        

          

            POST/api/customers/batchBatch create customers
          

          

            
curl -X POST http://localhost:5001/api/customers/batch \
  -H "Authorization: Bearer ***" \
  -H "Content-Type: application/json" \
  -d '{"count": 10}'

          

        

        

          

            POST/api/orders/batchBatch create orders
          

          

            
# Random customers
curl -X POST http://localhost:5001/api/orders/batch \
  -H "Authorization: Bearer ***" \
  -H "Content-Type: application/json" \
  -d '{"count": 5}'

# Linked to specific customers (by email)
curl -X POST http://localhost:5001/api/orders/batch \
  -H "Authorization: Bearer ***" \
  -H "Content-Type: application/json" \
  -d '{"count": 3, "link_customer_emails": ["john@gmail.com", "jane@example.com"]}'
